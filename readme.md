# README.md Learning File

This file demonstrates most common Markdown commands used in GitHub README files.

---

# 1. Headings

# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

Use `#` for headings.

Example:
```md
# Main Title
## Subtitle
```

---

# 2. Paragraph

This is a normal paragraph.

Markdown automatically joins lines together unless you leave an empty line.

Example:
```md
This is line one.

This is line two.
```

---

# 3. Bold Text

**This text is bold**

Example:
```md
**bold**
```

---

# 4. Italic Text

*This text is italic*

Example:
```md
*italic*
```

---

# 5. Bold + Italic

***Bold and italic together***

Example:
```md
***text***
```

---

# 6. Strikethrough

~~This text is crossed~~

Example:
```md
~~wrong text~~
```

---

# 7. Inline Code

Use `printf()` for printing.

Example:
```md
Use `code`
```

---

# 8. Code Block

```c
#include <stdio.h>

int main()
{
    printf("Hello World");
    return 0;
}
```

Example:
````md
```c
printf("Hello");
```
